# 🚀 LangChain Learning Repository

## 📌 Overview

This repository contains my learning journey of **LangChain**, a framework used to build applications powered by Large Language Models (LLMs).

LangChain helps in connecting AI models with external data, tools, and real-world use cases, making it easier to build intelligent applications like chatbots, Q&A systems, and AI assistants.

## 🎯 Why I Am Learning LangChain

I am learning LangChain to:

* Understand how modern AI applications are built
* Work with LLMs beyond basic prompts
* Build real-world projects like chatbots and RAG systems
* Improve my skills as a developer in the AI field
